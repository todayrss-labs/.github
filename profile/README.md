# TodayRSS Labs

TodayRSS Labs builds RSS infrastructure for Chinese and global content sources.

We help users and developers subscribe to important content sources through RSS, API and AI Agent integrations.

## Products

### 今天看啥

**今天看啥** focuses on RSS services for Chinese content sources.

Supported source types include:

- WeChat Official Accounts
- Bilibili
- Weibo
- Xiaoyuzhou
- Xueqiu
- RSSHub-based sources

Website: https://www.jintiankansha.me/

### TodayRSS

**TodayRSS** focuses on RSS services for global content sources.

Supported source types include:

- X / Twitter
- YouTube
- Instagram
- LinkedIn Company
- Telegram
- RSS Proxy

Website: http://cn.todayrss.com/

## Vision

Our goal is to turn important content sources into stable, subscribable and programmable feeds.

RSS is not only for readers.  
It can also become the information input layer for AI Agents, monitoring systems, research workflows and personal knowledge tools.
